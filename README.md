# ⚡ TaskFlow // Minimalist Planner

TaskFlow is a high-performance, minimalist task management application built with vanilla web technologies. It features a modern **Glassmorphism** UI, persistent data storage, and a mobile-responsive design.

![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)
![Version](https://img.shields.io/badge/version-1.0.0-success)

---

## 🚀 Features

* **Persistent Logic:** Tasks are saved to `localStorage`, ensuring your data survives page refreshes.
* **Modern UI:** A sleek dark-mode interface using Glassmorphism principles and CSS blur effects.
* **CRUD Operations:** Create, Read, Update (Toggle Status), and Delete tasks seamlessly.
* **Dynamic UX:** Auto-updating date header and empty-state handling when your list is clear.
* **Fully Responsive:** Optimized for desktop, tablet, and mobile screens.

## 🛠️ Tech Stack

* **HTML5:** Semantic structure for accessibility.
* **CSS3:** Custom variables, Flexbox, and keyframe animations.
* **JavaScript (ES6+):** Functional state management and DOM manipulation.
* **Storage:** Browser LocalStorage API.
